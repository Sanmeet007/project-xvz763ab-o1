# DocWise

An AI-powered prescription assistant that helps users understand medical prescriptions, compare available medicines across pharmacies, and follow clear, prescription-based medication schedules with timely reminders.


## Setup
.env.local file:

```env
GEMINI_API_KEY=<YOUR KEY>
```


## Server Starting command :

```bash
uvicorn app.main:app --reload
```
